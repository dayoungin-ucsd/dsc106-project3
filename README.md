# Degrees of Change: How Much Can Our Planet Handle?

Interactive visualization exploring thermal anomaly detections across U.S. states from 2020–2024 using NASA MODIS data.

## Live Demo
https://dayoungin-ucsd.github.io/dsc106-project3/

## Features
- Interactive U.S. choropleth map
- Year selector (2020–2024)
- Zoom and pan interactions
- NASA MODIS thermal anomaly data

## Data Source
NASA MODIS Thermal Anomalies / Fire Data
